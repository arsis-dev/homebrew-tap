# Arsis Homebrew Tap

Homebrew formulae for [Arsis](https://arsis.dev) developer tools.

## Installation

```bash
brew tap arsis-dev/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [sand](https://github.com/arsis-dev/sand) | Multi-backend workspace manager for developers (Zellij/tmux/Ghostty) |

### Install sand

```bash
brew install arsis-dev/tap/sand
```

Or in one command (tap + install):

```bash
brew install arsis-dev/tap/sand
```
